## Install on Chrome (Local / Unpacked)

1. Download or clone this project to your computer.

2. Open Google Chrome and go to:

3. Enable **Developer mode** (toggle in the top-right corner).

4. Click **Load unpacked**.

5. Select the project folder (the folder containing `manifest.json`).

6. The extension will appear in the Chrome toolbar.

---

## Grant site access (required)

1. Open any normal website (e.g. https://wikipedia.org).
2. Click the extension icon.
3. Click **“This can read and change site data”**.
4. Select **“On all sites”**.

---

## Use

- Open a web page.
- Click the extension icon to open the side window.
- FIRST Use **Select content from page** THEN **Summarize the page**.

---

## Notes

- The extension will **not work** on:
- `chrome://*`
- `chrome-extension://*`
- Chrome Web Store pages
